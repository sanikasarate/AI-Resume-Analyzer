AI-Resume-Analyzer

AI-Resume-Analyzer is a Python-based application designed to analyze resumes using AI and provide insights, match candidates with job descriptions, and generate scoring based on relevant skills and experience.

Features

Parse and process resumes in PDF or DOCX formats.

Extract key information like skills, education, and experience.

Match resumes against job descriptions using AI/ML algorithms.

Generate a score or ranking for candidate suitability.

Interactive web interface built with HTML, CSS, JavaScript, and Tailwind CSS.

Backend powered by Python, integrating AI models and APIs.

Tech Stack

Frontend: HTML, CSS, JavaScript, Tailwind CSS

Backend: Python (FastAPI, Django), AI/ML modules

Databases: PostgreSQL, MongoDB

Cloud & Deployment: AWS, Firebase, Hugging Face

Libraries & Tools: Pandas, NumPy, OpenAI GPT, BERT, RoBERTa

How to Use

Clone the repository:

git clone https://github.com/sanikasarate/AI-Resume-Analyzer.git


Set up a virtual environment and install dependencies.

Run the backend server with FastAPI:

python -m uvicorn src.api:app --reload


Open the frontend in your browser to upload and analyze resumes.
